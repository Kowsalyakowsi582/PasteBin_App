# Pastebin Lite

A simple Pastebin-like web application where users can create and share text pastes with optional expiry.

## Features

- Create and share text pastes
- Optional expiry time
- Optional view limits
- Responsive UI

## Tech Stack

- Next.js
- PostgreSQL (Neon)
- Tailwind CSS
- Vercel

## Running Locally

1. Clone the repo
2. npm install
3. Create .env.local and add DATABASE_URL
4. npm run dev

## Deployment

Deployed on Vercel using Neon PostgreSQL.
